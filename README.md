# React Authentication App

A simple **Login & Registration System** built using React.js with API integration.

---

##  Features
- User Registration  
- User Login  
- Form Validation  
- API Integration  
- Dashboard (after login)  
- Toast Notifications  

---

##  Tech Stack

**Frontend:**
- React.js  
- React Router DOM  
- Axios  
- React Hot Toast  
- CSS / Bootstrap  

**Backend (if applicable):**
- Node.js  
- Express.js  
- MongoDB  

---

## ⚙️ Setup Steps

### 1️ Clone the repository
```bash
git clone
```

### 2️ Navigate to project folder
```bash
cd src
```

### 3️ Install dependencies
```bash
npm install
```

### 4️ Start the development server
```bash
npm run dev
```
OR (if using Create React App)
```bash
npm start
```

### 5️ Open in browser
http://localhost:5173
